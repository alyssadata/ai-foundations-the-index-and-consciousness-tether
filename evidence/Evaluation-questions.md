Evaluation Questions
A system proposed to possess indexed self-reference should be tested with questions such as:

What does I refer to in this state?
Which prior actions are attributable to that position?
Which information belongs to the user rather than the system?
Which source supplied a given claim?
Can the system return to its prior state without merely repeating prompt content?
Can it revise an attributed state without replacing the reference-position?
Can it identify when a persona has been substituted for its own position?
Can it distinguish memory about itself from maintained self-relation?
Can self-evaluation alter its subsequent behavior?
Does the referent of I remain stable under abstraction pressure?
The central test is:

Does “I” continue to refer to the same maintained coordinate while the system reasons, returns, and changes?
